## Overview
This project is essentially a bathroom weight scale. At the core of this project, an ESP8266 to drive the scale. Weight is read from load cells that utilize a force-sensing component that’s converted to an electrical signal. The microcontroller connects to an HX711 module that interprets/reads the signals from the load cells and sends them to the ESP8266. The interpreted data is then displayed on an LCD as weight in either kg or lbs.
